Kamu adalah seorang mahasiswa IT yang baru memulai perjalanan, kemudian kamu juga sudah belajar tentang adanya pseudocode. 
Kemudian dosenmu memberikan sebuah tugas untuk membuat sebuah pseudocode tentang cara kerja sebuah thermometer yang akan merubah semua jenis suhu menjadi celcius.

Berikut adalah 3 cara konversi suhu ke dalam celcius:

fahrenheit to celcius = (N - 32) * (5/9)
kelvin to celcius = (N - 273.15)
celcius to celcius = N

JAWABAN 

Judul : Cara kerja thermometer dalam merubah semua jenis suhu menjadi celcius

Deklarasi :
  f = fahrenheit;
  k = kelvin
  c = celcius
  N = besar temperatur

  Input : 86 ( farenheit )
  Input : 45 ( celcius )

Deskripsi :

     f -> c = (N - 32) * (5/9)
     k -> c = (N - 273.15)
     c -> c =  N

     print(f) = 30c
     print(c) = 45c

END
